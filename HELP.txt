FOUR-HUB v1.0.0-ARCHITECT - Offensive Command Center
======================================================================

USAGE:
  four-hub [OPTIONS]

GLOBAL CONTROLS:
  F1 - F5         Switch primary views (Dashboard, Launcher, Workspace, Inspector, Terminal)
  t               Quick-set target (IP, Domain, or URL)
  r               Run targeted tool / Execute command
  S               Tactical Stealth Menu (Engage Shadow Mode)
  w               Automation Workflows
  ^e              Universal Export (HTML/JSON/Markdown)
  q               Secure Exit (Auto-wipe)

PRIMARY VIEWS:

  [F1] DASHBOARD
       Session heartbeat, live-feed activity monitoring, intelligence suggestions,
       and vulnerability severity breakdown.

  [F2] LAUNCHER
       The Arsenal. 160+ tools categorized. 
       Use [/] to search. [Tab] to cycle configuration fields.
       Target hints show required input format for each tool.

  [F3] WORKSPACE
       Strategic map of the target environment.
       Cycle panels (Hosts, Ports, Findings) with [Tab].
       [Enter] on a host to view port map. [Enter] on a finding to inspect.

  [F4] INSPECTOR
       Forensic analysis of scan results. 
       Syntax highlighting for banners. Scroll with [Up/Down] or [PgUp/PgDn].
       Export individual findings with [e].

  [F5] TERMINAL
       Integrated pseudo-terminal (Pseudo-TTY).
       Execute system commands within the encrypted environment.
       Command history is stored within the encrypted vault.

OPERATIONAL SECURITY (Stealth Menu):
  - 0. Shadow Mode: Enables all anti-forensics layers.
  - 1. Forensic Wipe: Securely overwrites temporary session data.
  - 2/3. MAC Shuffler: Randomizes NIC hardware identifiers.
  - 4. Identity Spoof: Renames the process to a standard kernel worker.

CONFIGURATION & PERSISTENCE:

  Configuration path: ~/.config/four-hub/config.toml
  Plugin directory:  ~/.local/share/four-hub/plugins/
