USAGE:
  four-hub [OPTIONS]

GLOBAL CONTROLS:
  F1 - F5         Switch primary views (Dashboard, Launcher, Workspace, Inspector, Terminal)
  t               Quick-set target (IP, Domain, or URL)
  r               Run targeted tool / Execute command
  S               Tactical Stealth Menu (Engage Shadow Mode)
  w               Automation Workflows
  ^e              Universal Export (HTML/JSON/Markdown)
  q               Secure Exit (Auto-wipe)

PRIMARY VIEWS:

  [F1] DASHBOARD
       Session heartbeat, live-feed activity sparkline, intelligence suggestions,
       and vulnerability severity breakdown.

  [F2] LAUNCHER
       The Arsenal. 160+ tools categorized. 
       Use [/] to search. [Tab] to cycle configuration fields.
       Target hints show required input format for each tool.

  [F3] WORKSPACE
       Strategic map of the target environment.
       Cycle panels (Hosts, Ports, Findings) with [Tab].
       [Enter] on a host to see port map. [Enter] on a finding to inspect.

  [F4] INSPECTOR
       Forensic analysis of scan results. 
       Syntax highlighting for banners. Scroll with [↑/↓] or [PgUp/PgDn].
       Export specific findings with [e].

  [F5] TERMINAL
       Integrated pseudo-terminal (Pseudo-TTY).
       Execute arbitrary OS commands inside the encrypted enclave.
       Command history saved in the encrypted vault.

ELITE OPS (Stealth Menu):
  • 0. Shadow Mode: Fully engages all 10 layers of anti-forensics.
  • 1. Forensic Wipe: Overwrites and deletes lingering temp traces.
  • 2/3. MAC Shuffler: Changes NIC hardware identifier.
  • 4. Personality Spoof: Changes process name to [kworker].

CONFIG & VAULT:
  Your session is protected by AES-256 encryption.
  Config location: ~/.config/four-hub/config.toml
  Plugins: Drop .py scripts into ~/.local/share/four-hub/plugins/
